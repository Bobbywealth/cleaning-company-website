## 3. Invoices - Email Delivery Fix

**Current State:** The "Send" button (line 336) just shows an alert - no actual email is sent.

**Changes needed:**
- Add email API endpoint to backend (`server/index.js`)
- Integrate email service (Nodemailer with SMTP or SendGrid)
- Update frontend "Send" button to call the API

**Backend (`server/index.js`):**
1. Add `nodemailer` dependency
2. Create `/api/invoices/send` endpoint
3. Configure SMTP transport from environment variables
4. Send HTML invoice email with invoice details

**Frontend (`Invoices.jsx`):**
1. Update "Send" button to call `POST /api/invoices/send` with invoice ID
2. Show loading state while sending
3. Show success/error notification

**Required Environment Variables:**
```
SMTP_HOST=smtp.example.com
SMTP_PORT=587
SMTP_USER=user@example.com
SMTP_PASS=password
EMAIL_FROM=360cleaning@company.com
```
